### 기능목록 구현 
- 입력
  - [ ] 차 이름을 ,를 구분자로 하여 입력받음 
  - [ ] 몇 번 이동할 것인지 입력받음
- 출력 
  -  [ ] 각 차수마다 자동차의 이동 결과
  - [ ] 최종우승자 ( 단독 우승자, 공동 우승자)
- 기능
  - [ ] 자동차 이름 입력받기
  - [ ] 진행할 회차 횟수 입력받기
  - [ ] 0~9 사이의 숫자를 랜덤으로 반환
  - [ ] 구분자를 기준으로 차를 분리(리스트? 맵?)
  - [ ] 반환받은 숫자가 4 이상이면 전진, 아니면 정지
  - [ ] 해당 회차에 경주 기록 저장 
  - [ ] 제일 많이 전진한 자동차 선택
  - [ ] 회차마다 전진된 누적 기록 보여주기
  - [ ] 우승자 출력 
  - [ ] 예외처리

### 예외처리

- [ ] 차의 이름을 입력하지 않음 
- [ ] 이동 횟수를 입력하지 않음 / 0임
- [ ] 횟수를 양수로 입력하지 않음
- [ ] 이름이 5자 이하가 아닐 때
